关于本站你需要知道的一切

发表于2024-11-10|更新于2024-11-14
|浏览量:

千呼万唤始出来
述职会议上提出的长期规划终于落地。一开始是想用jekyll驱动本站的，但是jeckll驱动过于困难且效果不好看，改用hexo为基础，butterfly美化的组合作为基本驱动。在挂载网站的选择上，经过多方比对，最终选择了使用github部署网站 gitee部署资源。
如果你出现github无法访问的情况，可以参考 不使用第三方工具访问 GitHub

众人拾柴火焰高
本站采用CC BY-NC-SA 4.0 许可协议进行知识、资源的分享。本站的生命力源于资源的丰富，但资源不是天上掉下来的。在班委成员的努力下，网站的框架成功搭建起来，并有了几门课的资源，但这对于一个长期运营的项目而言，远远不够。
WE NEED YOU
如果你有网站维护的意愿，有合理的想法诉求，有想要共享的资源，请联系我或对本项目提出 ***issue***。

你该如何使用本站
本站资料和资源分享均通过帖子实现，可以通过顶部导航栏 搜索 功能直接搜索，也可以通过 标签 功能实现按科目检索，或者通过分类 功能实现按学期检索。
进入贴文后你应该会发现一些经验（也可能不存在）和一个gitee的资源链接（一般表现为蓝色大号斜体超链接），里面是按照门类整理好的资源，选择你需要的门类直到出现pdf、ppt等你真正需要的终端资源，然后点击硕大的 下载 按钮，就可以静等资源下载完成了

大学物理
本文是大物的心得体会1、上课认真听讲，按需做笔记，比如在课本上标出相应内容，方便考前掌握重点考试内容。2、认真做作业，做时不看答案，做完后自行订正，能及时获得反馈。3、考前大约提前一周复习，把课本or笔记复习一遍，可以用一张纸总结出大物的所有公式，方便记忆。然后过错题，刷往届试卷，至少两套。这些步骤都做完就大物就差不多啦！Ps:其他科目也是一样的流程From: wjy
